# WebUni Online Tutoring Platform

Welcome to the WebUni Online Tutoring Platform!  
This project is designed to provide a comprehensive solution for online education, supporting multi-signup and login access for teachers, students, and parents.

## Features

- **Multi-Signup and Login Access**
  - Separate registration and login for Teachers, Students, and Parents
- **Attendance Management**
  - Teachers can mark and track student attendance
- **File Uploads**
  - Users can upload and share files and resources
- **Meeting & Call Scheduling**
  - Schedule and manage online meetings and calls between teachers, students, and parents
- **Role-Based Dashboards**
  - Tailored interfaces and access based on user type (Teacher, Student, Parent)

## Usage

- **Teachers:**  
  Register, log in, manage classes, mark attendance, upload resources, schedule meetings.
- **Students:**  
  Register, log in, attend classes, download resources, view attendance, join scheduled meetings.
- **Parents:**  
  Register, log in, monitor student progress, communicate with teachers, join meetings.
